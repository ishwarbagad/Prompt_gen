01
🔧 SELECT YOUR MCU
Go to the MCU tab. Search by name or filter by family. Click to select your microcontroller. You can also add a custom MCU not in the list using the Add Custom MCU form.
💡 Try searching "STM32F407" or "ESP32" — multi-word search works too like "STM32 motor"
02
⚡ SELECT PERIPHERALS & SENSORS
Go to Peripherals tab → select GPIO, ADC, PWM, DMA etc. Go to Sensors tab → select sensors like MPU6050, ACS712, DHT22. Add custom sensors if needed.
💡 Select Protocols in the Protocols tab — CAN, UART, I2C, MQTT etc.
03
📝 DESCRIBE YOUR PROJECT (Optional)
Go to Requirements tab. Write what your system does and list specific requirements — sampling rate, thresholds, data format etc. This makes the AI code much more accurate.
💡 Example: "Read ACS712 every 50ms. Alert if current > 5A via GPIO. Send JSON over UART at 115200."
04
⚡ CLICK "GENERATE CODE" — GET CODE DIRECTLY
Click the ⚡ GENERATE CODE button at the bottom. Choose your preferred AI platform:

• Generate Here (Claude AI) — Code generated directly on this page ✅
• ChatGPT / Gemini / Copilot — Prompt is copied, platform opens in new tab
💡 Use "Generate Here" for fastest results — code appears directly on the page with a copy button!

MCU SelectedSTM32F407VG · Cortex-M4F · 168MHz
FrameworkSTM32 HAL (CubeIDE)
SensorsACS712-20A (Current) + MPU6050 (Vibration) + DS18B20 (Temp)
ProtocolsI2C Master + UART DMA + MQTT

RequirementRead sensors every 100ms. Alert if current >5A. Send JSON via MQTT to ThingsBoard.
